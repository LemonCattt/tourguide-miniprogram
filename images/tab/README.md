# Tab Bar 图标说明

请将以下图标放置在本目录下，建议使用 81x81 像素 PNG 格式：

### 首页图标
- `home.png` - 首页（未选中），灰色 #999
- `home-active.png` - 首页（选中），蓝色 #1A73E8

### 发票图标
- `invoice.png` - 发票（未选中），灰色 #999
- `invoice-active.png` - 发票（选中），蓝色 #1A73E8

### 分房图标
- `room.png` - 分房（未选中），灰色 #999
- `room-active.png` - 分房（选中），蓝色 #1A73E8

### 我的图标
- `mine.png` - 我的（未选中），灰色 #999
- `mine-active.png` - 我的（选中），蓝色 #1A73E8

### Logo 图标
- `logo.png` - 应用 Logo（登录页使用）
- `default-avatar.png` - 默认头像

### 功能图标
- `icon-invoice.png` - 发票功能图标
- `icon-room.png` - 分房功能图标
- `icon-more.png` - 更多功能图标

> 建议使用 iconfont 或 Figma 设计图标后导出
